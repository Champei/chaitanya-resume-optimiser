# chaitanya-resume-optimiser
It analyzes any resume against a job description, extracts skills, experience, projects, compares them semantically (not just keywords), grounds the evaluation using real GitHub engineering language, and produces a trustworthy ATS score plus human-readable insights.
Resume (PDF/DOCX/TXT) -> Text Extraction -> ATS Engine (deterministic, factual) -> CrewAI (reasoning + explanation) -> Final Output
